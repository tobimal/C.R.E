# C.R.E(Create・Read・Evaluate)

## サイト概要
-創作物（小説、可能であればイラストなども）投稿サイト

### サイトテーマ
-作品に関わる全ての人が楽しむこと

### テーマを選んだ理由
-創作物投稿サイトを友人が利用しているため
また自分もファンとして楽しんでいるため

### ターゲットユーザ
-創作物を見てもらいたい人
-創作物を見たい人


### 主な利用シーン
-創作物の投稿、閲覧
